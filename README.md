# ITS - Doc bien so xe
Final-term assignment for the Intelligent Transportation Systems (ITS) course focusing on the development of a vehicle license plate recognition application.

How to run:
1. Download the file ZIP
2. Extract all
3. Open and check all the file path if they are correct or not as well as the library
4. Run complier
5. To run image: Type `python Image_test2.py`
6. To run video: Type `python Video_test2.py`
